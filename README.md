# Android Car Navigation Projects

This repository contains **14 complete starter Android projects** for your 2024 Hyundai Kona (Android Auto via Bluetooth).

## Project 1: Google Maps + Android Auto (Recommended for your Kona)
7 versions including:
- v1-simplest
- v3-gohome (one-click Go Home button)

## Project 2: OpenStreetMap + Bluetooth sync
7 versions including v1 and v3-gohome.

## How to use on your phone
1. Download the ZIP from this repo
2. Open any version folder in Android Studio (desktop) or AndroidIDE on phone
3. Build APK
4. Install on phone and test with Android Auto on your Kona.

More detailed instructions and code coming in the project folders.
